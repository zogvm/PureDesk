# PureDesk

# 这是啥？ 
 
 小米系统 自带的 极简桌面 纯净版

# 初心何在？

 制作的原因是 家人被商城的自动升级，升级到了3.1.28版之后，
 
 每次按下传统三键模式（左边按钮是当前运行的程序，中间是回到桌面，右边是返回到上一级）的中间按钮时，会弹广告。
 
 每按一下弹一次！
 
 要是不小心点到广告就下载各种垃圾应用！
 
 太可怕了！

 于是，我决定修改它，给家人一个纯净的桌面。
